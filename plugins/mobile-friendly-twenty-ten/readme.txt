Mobile Friendly Twenty Ten helps make websites and blogs running the Twenty Ten Theme - responsive.

== Description ==

Before downloading, it is obligatory that you read carefully the installation instructions!...
Mobile Friendly Twenty Ten helps make websites and blogs running the Twenty Ten Theme - responsive.
This is done by installing and moving from the Twenty Ten theme to the Twenty Eleven theme, which by installing the
Mobile Friendly Twenty Ten plugin is made to much look like the Twenty Ten theme.
Since the Twenty Eleven theme (that is now modified) is fully responsive, the resulting website is asort of 
a responsive Twenty Ten theme.

== Installation ==

1. Upload and activate the plugin through the 'Plugins' menu in WordPress
2. If changes have been made directly to PHP or CSS files, you should not continue the installation beyond this point without 
the assistance of an expert that would backup your website and update those changes after step 3, back to your new theme's files.
3. Upload and select the Twenty Eleven theme through the 'Design' menu and 'Themes' submenu in WordPress.
